class qiime.core.callable.Results¶

Bases: tuple

Tuple class representing the named results of a Callable.

Instances of this class may be returned by callables that have multiple return values, and are compatible with how Python handles multiple return values from callables. Provides an interface similar to a namedtuple type (e.g. fields are accessible as attributes).

Users should not instantiate this class directly.
